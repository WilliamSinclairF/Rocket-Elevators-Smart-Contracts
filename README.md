# how to run

- npm install
- check commands below
  
## Commands

- Compile:
  - `truffle compile`
- Migrate:
  - `truffle migrate`
- Test contracts:
  - `truffle test`
- Test dapp:
  - `cd app && npm test`
- Run dev server:
  - `cd app && npm run start`
- Build for production:
  - `cd app && npm run build`
